# Library Management System

## Features
- Role-based authentication (Admin, Librarian, User)
- JWT Authentication
- Password hashing using bcrypt
- Book issue tracking
- Top 3 ranked books system
- Secure HTTP headers
- CORS enabled

## Tech Stack
Frontend: React
Backend: Node.js, Express
Database: MongoDB

## Security
- Password hashing
- JWT authentication
- CORS protection
- Server-side validation
- Secure HTTP headers using Helmet
